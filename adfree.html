> CUM SĂ SCAPI DE RECLAME
(O soluție pentru reclamele de pe server-ul Filemoon)
- Pentru PC (Windows):
1. Folosește browser-ul Chrome.
2. Accesează Magazinul Web Chrome: Link aici
3. Descarcă următoarele extensii:
- "Adblock Plus" - Link aici
- "AdBlock - cel mai bun blocant de reclame" - Link aici
- "uBlock Origin" - Link aici
4. Pentru extensiile care deschid pagini noi atunci când le instalezi (ex.: cerând donații, etc.), nu-ți face griji! Pur și simplu închide pagina nou deschisă și bucură-te de site-urile preferate fără reclame!
- Pentru telefon (Android/iOS):
1. Descarcă din Magazin Play sau App Store aplicația "Kiwi Browser".
2. Deschide noua aplicație.
3. Accesează Magazinul Web Chrome: Link aici
4. Descarcă următoarele extensii:
- "Adblock Plus" - Link aici
- "AdBlock - cel mai bun blocant de reclame" - Link aici
- "uBlock Origin" - Link aici
5. Pentru extensiile care deschid pagini noi atunci când le instalezi (ex.: cerând donații, etc.), nu-ți face griji! Pur și simplu închide pagina nou deschisă și bucură-te de site-urile preferate fără reclame!

*Extensiile vor avea efect pentru toate site-urile pe care le vei accesa din Chrome/Kiwi browser.
